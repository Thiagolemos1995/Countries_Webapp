# Countries_Webapp

 Projeto de uma página web que consome uma API [Rest Countries](https://restcountries.eu/).
 
 ## Descrição
 
 A página inicial do app dispõe todos os países do globo, sendo possível aplicar filtros para otimizar a busca.
 
 ## Instruções
 
 A primeira caixa de seleção, lhe permite acessar os tipos de filtros disponível, seja por Região, Capital, Lingua, Moeda ou Código de ligação.
 Ao selecionar a região é mostrada uma segunda caixa de seleção com as regiões disponíveis e ao clicar em pesquisar ele irá mostrar todos os países pertencentes aquela região.
 Ao selecionar Capital é mostrado um campo de texto que permite que você escreva o nome da capital do país em questão para que ao clicar em pesquisar ele mostre o país que aquela capital pertence.
 Ao selecionar Linga é mostrado também um campo de texto, onde o filtro é aplicado utilizando a norma ISO 639-1 para buscar pelo país.*
 Ao selecionar Moeda é mostrado também um campo de texto, onde o filtro é aplicado utilizando a norma ISO4217 para buscar pelo país.*
 Ao selecionar Código de Ligação é mostrado um campo numérico onde o filtro é aplicado de acordo com o código de área referente a cada país.
